# TodoList
Simple ToDo List app for iOS

function: creating todo, checkmark, reordering, deleting, renaming

<img src="https://github.com/khhk10/TodoList/blob/master/images/todolist.jpg" width="700">

### Tools
- UITableView
- NSKeyed(Un)Archiver
- UserDefaults
- UIAlertController

### Requirement
- iOS 12.1
- Xcode 10.1
