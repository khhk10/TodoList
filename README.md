# TodoList
Simple ToDo List app for iOS

<img src="https://github.com/khhk10/TodoList/blob/master/images/todolist.jpg" width="700">

- create todo
- edit (reorder, rename, delete)
- checkmark

### Tools
- UITableView
- NSKeyed(Un)Archiver
- UserDefaults
- UIAlertController

### Requirement
- iOS 12.1
- Xcode 10.1
